Grand principe

  • Créé un fichier CSS externe à la page HTML
  • Relier ce fichier CSS au fichier html

    • Comment faire
    1. Aller dans le < head > du fichier html
    2. Utiliser la balise < link rel="stylesheet" href="Chemin_du_fichier_CSS" >

  • CSS est capable d'identifier tout les éléments HTML grâce à un selecteur : ici p

    • exemple

    • Pour le paragraphe du fichier html ( < p > )

      •                 p
                {
                    < propriété > : < valeur > ;
                    < propriété > : < valeur > ;
                    < propriété > : < valeur > ;
                }
    • On attribue pour les paragraphes du fichier HTML relier, un style définie dans le selecteur p

  • Le selecteur peut s'appliquer à une partie seulement du fichier HTML : avec les ID et les Classes

    • exemple
    • Imaginons que nous avons créé une id="test"dans notre deuxième paragraphe du fichier html
    • Pour pouvoir faire référence à cette id il faut indiquer ceci :
      •                 #test ( Tout ce qui aura pour identifiant : test, auront ces modifications )
      
                ou bien
      
                p#test ( Les paragraphes qui ont l'identifiant test, auront ces modifications )
      
                {
                    < propriété > : < valeur > ;
                    < propriété > : < valeur > ;
                    < propriété > : < valeur > ;
                }
    • S'il s'agît d'une classe, il faut indiquer ceci :
      •                     p.test

  • On attribue pour les paragraphes du fichier HTML relier, un style définie dans le selecteur

Modèle de boite

  • En html, les < body > , < p > , < div > , < html > et autres, fonctionne par un système de boite.

  • Dans le fichier CSS, nous pouvons donner une dimension à ces boites

    • body{ ;}

  • Content (element) / Padding / Border / Outline / Margin


  • Modifier les marges interieur et exterieur d'une boite

    • body{ ;}


    body{ ;}

  • Gérer les dépassement de boite

    • body{overflow : ;}

    body{overflow-x-y : ;} Ne s'applique que pour l'axe x ou y

    body{overflow-wrap: ;} Gère le paragraphe lui même

  • Appliquer une ombre à sa boite ( box-shadow peut prendre plusieurs valeurs : Horizontale, Verticale, portée de l'ombre, Couleur )

    • div { box-shadow : ;}

  • Par défaut, la largeur d'une boite conteneur ne prend pas en compte les eventuelles dépassement de largeur de leurs contenu avec padding, border


    • Pour prend en compte ces eventuelles dépassements

    div { box-sizing : ; }

Bordures et contours

  • Appliquer une bordure ( L'attribut border prend 3 élément : border : witdh ( épaisseur ), style ( le style ), color ( la couleur ) )

    • body { border : ;}

  • Appliquer une image à sa bordure

    • ( L'attribut border-image prend : source ( chemin de l'image ), slice ( découpe de l'image ), witdh ( épaisseur ), outset ( marge ), repeat ( répétition de l'image ) ) L'image utiliser est découper en 9 parties. La partie central de la découpe est l'élément central ou s'applique la bordure.

  • Lien pour généré sa bordure à partie d'une image


  • Gérer les arrondies des bords de la bordure

    • body { border-radius : ;}

    On peut cibler des coins précis en utilisant la méta attribut : { border-radius : "top-left" "top-right" "bottom-right" "bottom-left" }

  • Pour le contours, les attributs sont les mêmes que pour border, elle s'utilise avec

    • body { outline : ;}

  • Appliquer une marge entre border et outline

    • body { outline-offset : ;}

  • Collision des bordure d'un tableau

    • table { border-collapse : ;}

  • Espacement des bordures d'un tableau

    • table { border-spacing : ;}

Affichage et positionnement

Affichage

  • Balise type Block et Inline

    • Block

    • < div >
    • < p >
    • < h1..2..3.. >
    • < ul..ol >
    • < blockquote >

    Inline

    • < span >
    • < a >
    • < strong >
    • < em >
    • < br >
    • < img >


  • Comportement Bloc et Inline

    • Block

    • Utiliser pour structurer le contenu en sections distinctes
    • Elles peuvent avoirs des propriété de largeur ( witdh ), hauteur ( height ), margin, padding
    • Occupe toute la largeur disponible
    • Commence sur une nouvelle ligne
    • Pousse les éléments suivant vers le bas

    exemple

    Ce text est en réalité

    3 < p > qui sont mis

    les uns à la suite des autres


    Inline

    • Utiliser pour envelopper et styliser des parties spécifiques de texte
    • Occupe uniquement l'espace nécessaire autour de leurs contenu
    • Ne commence pas sur une nouvelle ligne
    • S'affichent sur la même ligne que le contenu qui les entoure

    exemple
    Ce text est en réalité 3 < span > qui sont mis les uns à la suite des autres

  • Changer le type d'une balise

    • span { display : ;}

  • Cacher un contenu

    • span { visibility : ;}

  • Opacité du contenu par défaut

    • img { opacity : 0..1 ;}

  • Opacité du contenu au passage de la souris

    • img : hover { opacity : 0..1 ;}

Flex-box

  • Illustration Flex-Box


  • Organise des éléments à petite échelles


  • Flex-box ne travaille que dans une direction à la fois


  • Changer le type d'une balise structurante en flex-box ou inline-box

    • div { display : ;}

  • Modification du comportement du conteneur


    • Changer la direction du conteneur ( Définie l'axe principal et secondaire, row = Alignement principal horizontale, column = Alignement principal verticale )

      • div { flex-direction : ;}

    • Gérer la coupure du contenu

      • div { flex-wrap : ;}

    • Attribut qui rassemble les deux propriété : div { flex-flox : "direction" "coupure" ;}


    • Gérer l'alignement principal du contenu ( Ce qui définie l'alignement principal du conteneur flexible est sont flex-direction, row = alignement principal horizontal, column = Alignement principal verticale )

      • div { justify-content : ;}

    • Gérer l'alignement secondaire du contenu

      • div { align-items : ;}

  • Modification du comportement du contenu


    • Gérer l'ordre du contenu ( objet ) Changer l'ordre permet d'indiquer sa prioritée dans la disposition à l'interieur du conteneur

      • div { order : ;}

    • Gérer l'occupation de l'espace du contenu

      • Gérer l'agrandissement

        • div { flex-grow : ;}

      • Gérer le rétrecissement

        • div { flex-shrink : ;}

      • Définir une largeur initial

        • div { flex-basis : ;}

      • Attribut qui rassemble les trois propriétés : div { flex : "agrandissement" "retrecissement" "largeur initial" ;}

      • Changer le comportement d'un contenu par rapport aux autres

        • div { align-self : ;}

Grid

  • Ilustration Grid


  • Organise des éléments à grandes échelles ( Le Gabarit )


  • Grid est bi-dimentionnelle


  • Changer le type d'une balise structurante en grid

    • div { display : ;}

  • Dans le conteneur

    • Indiquer un nombre de colonne

      • div { grid-template-columns : ;}

    • Indiquer un nombre de ligne

      • div { grid-template-rows : ;}

    • Indique le positionnement pour les éléments non définie explicitement

      • div { grid-auto-flow : ;}

    • Indiquer une largeur pour les éléments non définie explicitement positionner en colonnes

      • div { grid-auto-columns : ;}

    • Indiquer une hauteur pour les éléments non définie explicitement positionner en lignes

      • div { grid-auto-rows : ;}

  • Dans le contenu ( items )

    • Indiquer la position de l'item sur une colonne précise

      • div { grid-column : ;}

    • Indiquer la position de l'item sur une ligne précise

      • div { grid-row : ;}

    • Indiquer le point de départ de l'item sur une colonne précise

      • div { grid-column-start : ;}

    • Indiquer le point de fin de l'item sur une colonne précise

      • div { grid-column-end : ;}

    • Indiquer le point de départ de l'item sur une ligne précise

      • div { grid-row-star : ;}

    • Indiquer le point de fin de l'item sur une ligne précise

      • div { grid-row-end : ;}

  • Définir le découpage des airs de placements

    • Protocole

      • On attribut un nom d'airs à un selecteur :

        • div_1 { grid-area : "nom_1" ;}

        div_2 { grid-area : "nom_2" ;}

        div_3 { grid-area : "nom_3" ;}

      • On défini leurs emplacements Dans le conteneur :

        • div { grid-template-areas : " nom_1 nom_1 nom_1 "
                            " nom_2 nom_1 nom_2 "
                            " nom_3 nom_3 nom_3"


      • div_1 représentera les trois colonnes de la première ligne de la grid
      • div_2 représentera la colonne à gauche et à droite de la deuxième ligne de la grid, div_1 la colonne central de la deuxième ligne de la grid
      • div_3 représentera les trois colonne de la troisième ligne de la grid
