Variable en dynamique
Il est possible de récupérer une saisie utilisateur pour la stocker dans une variable.
par exemple avec prompt()

var age = prompt(" Quel âge as-tu ? ")

Avec prompt, un message va s'afficher avec la phrase : " Quel âge as-tu ? ",
Et la saisie de l'utilisateur sera stocker ensuite dans la variable age
Il est possible ensuite de l'utiliser, par exemple pour l'afficher :

document.write("Tu est agées de "+age+" ans");

+ sert pour la concaténation
Convertir le type d'une variable
par défaut, prompt() renverra toujours un string.
Si on veut pouvoir effectuer des calculs avec la variable age par exemple,
il faudra la convertir en number

age = parseInt(age)

A ce moment là, la variable age est devenue un Integer.

Les fonction de conversion :

parseInt() => Converti en Integer.
parseFloat() => Converti en Flottant.

Opérateur et opérateur de comparaison
Incrémentation Post / Pre
let number = 10;

Pré-incrémentation :

++Number;

( ce que ça fait : number + 1 et ensuite number = number )

Post-incrémentation :

Number++;

( ce que ça fait : number = number et ensuite number + 1 )

Exemple de différence :

document.write(number++); ==> 10

( Dans l'ordre, javascript va gérer l'affichage, puis l'incrémentation ).

document.write(++number); ==> 11

( Dans l'ordre, javascript va gérer l'incrémentation, puis l'affichage ).
Même valeur ET même type
document.write(8 === 8); ==> True

l'opérateur === renverra true si les deux élément sont de même valeur et de même type.
C'est le cas, en revanche, elle reverrais False dans ce cas :

document.write(8 === "8"); ==> False

La valeur est la même, mais le premier est de type Integer et le second est de type String

L'opérateur == vérifie simplement la valeur, et pas le type, elle aurait renvoyer True dans
différent type OU différente valeur
document.write(8 !== 8); ==> False

l'opérateur === renverra False si les deux élément sont de même valeur et de même type.
C'est le cas, en revanche, elle reverrais True si :

La valeur est différente OU le type est différent OU les deux. ( Dès que ce n'est pas strictement égal )

Par exemple :

document.write(8 !== 1); ==> True ( valeur différente )
document.write(8 !== "8"); ==> True ( Type différent )
document.write(8 !== "1"); ==> True ( valeur et type différent )

Fonction
Définir une fonction
function calcule(a, b)
{
return a + b;
}

let resultat = calcule(10, 5);

document.write(resultat);


Création de la fonction calcule avec deux paramètre obligatoire : a b
Ce que doit faire la fonction : retourner le résultat de a + b
Comme il y a un return, on recupère ce retour dans une variable : resultat
Il y avait deux paramètre obligatoire, on lui donne : 10 5 en argument lors de l'appel de la fonction
Enfin, on affiche le resultat avec document.write
Avoir des paramètre facultatif
Pour indiquer dans une fonction des paramètre facultatif, il faut leurs donner une valeur par défaut, quand elles ne sont pas indiquer dans l'appel de la fonction

function calcule(a, b = 5)
{
return a + b;
}

let resultat = calcule(10);

document.write(resultat);

Dans l'appel de la fonction, nous n'avons pas indiquer le paramètre b puisqu'il est faculatif
( il a une valeur par défaut de : 5 )

Il prendra donc pour le paramètre b la valeur par défaut, comme nous ne l'avons pas indiquer dans son appel

( si un argument avait été donner pour b dans l'appel de la fonction, cette argument aurait remplacer la valeur par défaut )
Nombre de paramètre infinie
function calcule( ...args )

Programmation objet
Principe
Le principe de la programmation objet est de créé des classes dans lesquelles nous allons pouvoir générer des objets
Ces objets auront des attributs, des methodes.

Par exemple :

Classe : Chien
Objet : Snoopy
Attribut : poils = fluffy, couleur = marron, yeux = trop mignon, etc
Methode : Aboie quand le facteur passe, sautille partout, mord s'il est pas content
Définir une classe
class Music_Player
{
    constructor(format)
    {
        this.format = format;
        document.write(this.format);
    }

    play()
    {
        consol.log("Lecture en cours");
    }
}

On a :
Définie une class : class Music_Player
Définie un constructeur : this.format = format
Définie une methode : play() ( une méthode est une fonction de classe )
Créé une instance de classe ( un objet )
let user_1 = new Music_Player("MP3");

let user_2 = new Music_Player("WAV");

user_1.play();

On a créé deux instances de classe : user_1 et user_2
Dans le constructeur, on a renseigner un attribut obligatoire : format, on le renseigne lors de la créationd e l'objet : "MP3" et "WAV"

On a appeler une methode de la classe Music_Player en indiquant un objet et sa methode : user_1.play()
Modifier un attribut de classe
Il n'est pas encore possible de rendre non accessible du global des attribut de classe, c'est pour cela qu'il est conseiller dans les methode de classe de procéder comme suit :

Créé deux methode, l'une capable de lire un attribut, l'autre de la modifier :

get_format()
{
return this.format;
}

set_format(new_format)
{
this.format = new_format;
}

Il suffira de les appeler : user_1.get_format(), user_1.set_format(nouveau_nom)

Méthode static
La méthode play() est accessible via l'instance de la classe Music_player :

song.play();

Une méthode static est une methode qui est liée à la classe elle même, et pas à une instance de classe,
elle utilise le mot clef static, dans la classe, il faut donc écrire :

static hello()
{
return "Hello";
}

Pour y accéder, il faudra passer par la classe elle même, et pas par une instance :

Music_player.hello()

Pour une méthode :

{
return "bye";
};

Créé un objet
Il existe plusieurs méthode pour créé un objet

Par une variable, une fonction, une classe
En passant par une variable :
let naruto = 
{
    name : "naruto",
    weapon : "shuriken",
    attack : function()
            {
                return "Rasengan !"
            }
}

On peut acceder aux attribut et valeurs de l'objet naruto :

document.write(naruto.attack());
document.write(naruto.name);
document.write(naruto.weapon);
En passant par une fonction :
function ninja(name, weapon)
{
    this.attack = function()
    {
        return "Rasengan !";
    }

    this.name = name;
    this.weapon = weapon;
}

let naruto = new ninja("naruto", "shuriken")

On créé un objet naruto en passant par le mot clef new
On peut acceder aux attribut et valeurs de l'objet naruto :

document.write(naruto.attack());
document.write(naruto.name);
document.write(naruto.weapon);
En passant par une classe :
