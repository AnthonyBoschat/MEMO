Fonctionnement de base

Il existe 2 type de balise :

  • Classique : Elles nécessitent une ouverture et une fermeture de balise.

    exemple : < p > (ouverture de la balise) < /p > (fermeture de la balise)

  • Orpheline : Elles ne nécessitent pas de fermeture de balise, une seule mention suffi.

    exemple : < br > (ne necessite pas de fermeture de balise)

Principe de fermeture de balise imbriqué :

  • On ferme toujours en premier la dernière balise qui a été ouverte (symétrie central) :

    exemple : < p > < strong > Message important < /strong > < /p >

Balise général

Commenté le code :

< ! - - Commentaire - - >

Nouveau paragraphe :

< p > Texte < /p >

Saut de ligne :

< br >

Séparateur (ligne noir) :

< hr >

Indiquer un texte important (par ordre d'importance) :

  1. < strong > Texte important ++ < /strong >
  2. < em > Texte important < /em >
  3. < b > Texte semi-important < /b >
  4. < i > Texte un peu hors propos < /i >

Texte obsolète :
Texte obsolète à remplacer :

< del > Ancien texte < /del >
exemple : Anthony à 27 ans

Texte qui le remplace :

< ins > Nouveau texte < /ins >
exemple : Anthony à 27 28 ans

Texte pas pertinent :

< s > Texte pas pertinent < /s >

Texte surligné

< mark > Texte surligné < /mark >

Titre (par ordre d'importance) :

  1. < h1 > Premier titre < h1 >
  2. < h2 > Deuxième titre < h2 >
  3. ...
  4. < h6 > Dernier titre < h6 >

Lien internet :
Sans infobulle :

< a href="lien" > Texte en lien < /a >
exemple : Google

Avec infobulle :

< a href="lien" title="texte de l'infobulle" > Texte en lien < /a >
exemple : Google

Ouvrir le lien dans un nouvel onglet :

< a href="lien" title="texte de l'infobulle" target="_blank"> Texte en lien < /a >
exemple : Google

Citation de source externe :
Sans la source :

< q > Citation < /q >

Avec la source :

< q cite="http..." > Citation < /q >
exemple : Abraham Lincoln

Citation d'une oeuvre, nom de document (pas de personne) :

< cite > Citation d'un nom < /cite >
exemple : Montargis

Citation longue :
Sans la source :

< blockquote > Citation longue < /blockquote >

Avec la source :

< blockquote cite="http..." > Citation longue < /blockquote >

Liste :
Non ordonnée (liste à puces) :

    < ul >
          < li > Premier élément de la liste < li >
          < li > Deuxième élément de la liste < li >
          < li > Troisième élément de la liste < li >
    < /ul >

exemple
  • Faire ceci
  • Ou ça
  • Ou bien cela

Ordonnée :

    < ol >
          < li > Premier élément de la liste < li >
          < li > Deuxième élément de la liste < li >
          < li > Troisième élément de la liste < li >
    < /ol >

exemple
  1. Faire ceci
  2. Puis ça
  3. Et enfin cela

Attribut possible pour la liste ordonnée
< ol Attribut>
  • reversed : Inverse la numérotation
  • start="10" : Commence la numérotation à 10
  • type="a" : Ordonne alphabétiquement
  • type="I" : Ordonne avec des chiffres romain

Sous-liste :

    < ul >
          < li > Un premier élément < /li >
          < li > Un deuxième élément < /li >
                < ol >
                      < li > Etape a du deuxième élément < /li >
                      < li > Etape b du deuxième élément < /li >
                      < li > Etape c du deuxième élément < /li >
                < /ol >
          < li > Un troisième élément < /li >
    < /ul >

exemple
  • Un premier élément
  • un deuxième élément
    1. Etape a du deuxième élément
    2. Etape b du deuxième élément
    3. Etape c du deuxième élément
  • Un troisième élément

Formulaire

Créé un formulaire :

< form > < /form >

Attribut de < form > :

    < form method="get"> (ajoute les donnée du formulaire sur l'URL de la page)
    < form method="post"> (Fourni les données avec l'envoi du formulaire)

    < form action=""> (Destination d'envoi du formulaire)
    < form name="Mon Formulaire"> (Donner un nom)
    < form target="_blank"> (Envoi du formulaire au niveau du contexte)
    < form autocomplete="on"> (Active l'auto-complétion du formulaire)
    < form autocomplete="off"> (Desactive l'auto-complétion du formulaire)
    < form enctype=""> (Spécifie un type MIME)
    < form accept-charset="UTF-8 etc..."> (Spécifie plusieurs encodage accepter)

Standard Form Element

Champ de saisie pour le formulaire

< input >

Attribut de < input >

    < label for="prenom"> Nom : < /label > < input id="prenom" name ="username"> (Accessibilité sans souris au formulaire)
    < input placeholder="Entrez votre nom..."> (Specifie un texte guide dans l'input)
    < input type=""> (Donne un type)
    < input required> (Indique qu'il faut remplir le champ)
    < input value=""> (Donne du texte coté client)
    < input nom=""> (Donne un nom utilisable dans le code)

Valeur de < input type > :

    < input type="texte"> (Génère du texte standard)
    < input type="password"> (Pour un mot de passe : saisie masquer)
    < input type="checkbox"> (case à cocher non obligatoire)
    < input type="radio"> (Case à cocher obligatoire)
    < input type="color"> (Gestion de couleur : Palette)
    < input type="date"> (Selection calendrier)
    < input type="datetime-local"> (Selection calendrier + heure/minute)
    < input type="month"> (Selection Mois/Année)
    < input type="week"> (Selection Semaine)
    < input type="email"> (Saisie d'email)
    < input type="tel"> (Saisie clavier numérique)
    < input type="search"> (Pour moteur de recherche)
    < input type="range"> (jauge déplacable) ( type="range" min="0" max="100" step="5")
    < input type="number"> (Aggrégation de nombre)
    < input type="hidden"> (formulaire caché)
    < input type="submit"> (Bouton de validation) (type="submit" value="Se connecter")
    < input type="reset"> (clear le formulaire)

              < img src="Chemin de l'image" >

Structurer une page web

Bloc de contenu / Division de bloc :

    < div > < /div >

Ligne de contenu / Division de ligne :

    < span> < /span >

Contenu principal du body :

    < main > < /main >

Sectionner une partie du document ( en général pourvu d'un titre ) :

    < section > < /section >

Bloc indépendant pas propre à la page ( qui pourrait être écris ailleurs ) :

    < article > < /article >

Entête d'un bloc / section / body :

    < header > < /header >

Pied d'un bloc / section / body :

    < footer > < /footer >

Menu de naviguation :

    < nav > < /nav >
exemple
  • Nom
  • Age
  • Adresse
  • Taille

Contenu quelconque / à part :

    < aside > < /aside >
exemple structure correcte 
< body >
        < main > ( Contenu principal du site )

                < header >
                < h1 > Titre / Entête principal du site < /h1 >
                < /header >

                < section >
                            < header >
                            < h1 > Titre / Entête de ma section < /h1 >
                            < /header >
                            < p > Contenu de ma section < /p >
                            < footer >
                            < p > Pied de ma section < /p >
                            < /footer >
                < /section >

                < footer >
                < p > Pied de mon site < /p >
                < /footer >

        < /main > ( fin du contenu principal du site )

        < p > Paragraphe qui n'est pas le contenu principal du site < /p >

< /body >
